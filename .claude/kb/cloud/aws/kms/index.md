## Key Concepts

| Concept | Description |
|---------|-------------|
| **KMS Key** | Logical key that never leaves KMS; used to encrypt data keys or up to 4 KB directly |
| **Data Key** | Plaintext key generated by KMS for client-side envelope encryption |
| **Envelope Encryption** | Encrypt data with data key, encrypt data key with KMS key |
| **Key Policy** | Resource-based policy on a KMS key controlling who can use/manage it |
| **Grant** | Temporary, revocable delegation of KMS key permissions |
| **Alias** | Friendly name (alias/my-key) that points to a KMS key ID |
| **ML-DSA** | Post-quantum digital signature key specs (FIPS 204) |
