## Why GitHub Actions?

GitHub Actions is a CI/CD platform built directly into GitHub, enabling automated workflows triggered by repository events. Its tight integration with GitHub makes it the natural choice for teams already hosting code on GitHub.

**When to use GitHub Actions:**
- CI/CD for GitHub-hosted repositories
- Automated testing on pull requests
- Container builds and registry pushes
- Release automation and deployment pipelines
