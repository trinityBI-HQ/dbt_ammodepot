## Why Kafka?

Apache Kafka is a distributed event streaming platform for high-throughput, low-latency data pipelines. It serves as the backbone for real-time data architectures, enabling event-driven microservices and stream processing.

**When to use Kafka:**
- Real-time event streaming between systems
- High-throughput log aggregation
- Event sourcing and CQRS architectures
- Stream processing with Kafka Streams or Flink
