## Lessons Learned

### Process

- **Brainstorm→Define→Design→Build in one session works for well-scoped features.** The 6 brainstorm questions and 2 validation checkpoints caught all major design decisions upfront. Zero rework on the core approach.
- **Validate assumptions before build, not during.** Testing `dbt docs generate --static` and the IAM policy locally saved CI debugging time. The one we didn't validate (SiS CSP blocking iframes) caused a post-deploy fix.

### Technical

- **`dbt docs generate --static` is the right choice for CI.** Produces a single 4.3 MB self-contained HTML — no multi-file hosting, no static website config. Available since dbt 1.7.0, works on 1.11.6.
- **S3 Block Public Access prevents `--acl public-read`.** Newer S3 buckets default to BucketOwnerEnforced. Presigned URLs are the clean workaround — no public access needed.
- **SiS CSP blocks external iframes.** `st.components.v1.iframe()` with external URLs shows "This content is blocked." Use `st.link_button()` to open in a new tab instead. (Saved to memory.)
- **`generate_presigned_url()` is a local signing operation.** No network call needed — don't pre-check with `head_object` which requires S3 egress.
- **CloudWatch Logs `get_log_events` returns per-stream.** Each ECS task run is a separate log stream. `describe_log_streams(orderBy='LastEventTime', descending=True)` gives reverse-chronological builds.

### Communication

- **User's suggestion of `--static` flag improved the design significantly.** Always ask about flags/features the user may know about — domain knowledge outweighs web search.
- **User's "use the same bucket" saved an entire infrastructure step.** Reusing `ammodepot-lakehouse` with a `dbt-docs/` prefix eliminated bucket creation, IAM policy changes, and A-002 validation.

### Tools & Libraries

- **boto3 `get_metric_data`** — cleaner than `get_metric_statistics` for time-series. Returns timestamps + values directly.
- **Plotly `add_hline`** — simple API for reference lines with annotations. The 10-min ceiling line is one call.
- **`st.link_button(use_container_width=True)`** — clean full-width button for external links in SiS.
